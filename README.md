# Markforged EMEA — Direct CPQ

Internal direct quoting tool for the EMEA sales team.

## What it does
A lightweight browser-based tool to configure, price, and generate customer quotes for Markforged products. Covers product selection, discount approval tiers, Digital Forge Complete subscriptions, shipping & handling, and quote export.

## Access
PIN protected — internal use only.

## Stack
Single HTML file · React 18 · No backend · No database
